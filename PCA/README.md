# PCA

● **Brief Description:** This folder contains a dimensionality reduction project based on Principal Component Analysis (PCA), including analysis in a notebook and required dependencies.

● **Project Objective:** Reduce feature dimensionality, analyze explained variance, and support downstream modeling or visualization tasks.

● **How the Files Are Organized:**
- `PCA.ipynb`: Main notebook with PCA workflow and analysis.
- `dataset.csv`: Local dataset used for PCA experimentation.
- `requirements.txt`: Python dependencies for this project.

● **Dataset Used:** `dataset.csv`.

● **Technologies Used:** Python, Jupyter Notebook, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, SciPy, Yellowbrick.

● **Key Results:** PCA transformations and explained-variance outputs are generated in `PCA.ipynb`.
