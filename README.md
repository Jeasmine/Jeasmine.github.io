# Data Science & Machine Learning Portfolio

This repository showcases end-to-end machine learning work across forecasting, NLP, computer vision, and dimensionality reduction. Each project combines practical modeling with structured experimentation in Jupyter notebooks, clear data organization, and reproducible Python environments.

## Why This Portfolio

I build ML solutions with a product mindset: frame the problem, explore the data deeply, test multiple modeling strategies, and keep artifacts organized so results can be reused and improved.

## Projects

### 1. Sales Forecasting
**Folder:** [`sales_forecasting/`](sales_forecasting/README.md)

- Built forecasting workflows using both classical ML and deep learning approaches.
- Compared gradient boosting models (XGBoost/CatBoost) against LSTM-based sequence models.
- Produced reusable trained model artifacts (`model_*.keras`) for category-level predictions.
- Maintained dataset, logs, and notebooks in a structure designed for iteration and traceability.

### 2. Customer Experience Classifier (NLP)
**Folder:** [`cx_classifier/`](cx_classifier/README.md)

- Developed a conversational text classification pipeline for CX-related data.
- Worked with multitable relational datasets (`conversations`, `messages`, `entities`, `labels`).
- Combined EDA, feature/model experimentation, and transformer-ready tooling.
- Organized versioned dataset snapshots to support reproducible training.

### 3. Skin Cancer Classification Experiments
**Folder:** [`skin_cancer/`](skin_cancer/README.md)

- Explored multiple modeling tracks: image-only baseline, multimodal (image + tabular), and transfer learning.
- Built notebook-driven experimentation flow for fast comparison of approaches.
- Integrated computer vision and structured data techniques in one project.

### 4. PCA / Dimensionality Reduction
**Folder:** [`PCA/`](PCA/README.md)

- Implemented Principal Component Analysis workflow for feature-space simplification.
- Evaluated explained variance and component behavior to support downstream modeling.
- Demonstrated interpretable preprocessing for high-dimensional datasets.
